# Chat CSS

The `themes` directory here is a symlink to the original files in `/core/static/core/css/chat/themes/`.

This structure is maintained for backward compatibility. For all new development, please use the files in the core app's static directory.

## Structure

- `/core/static/core/css/chat/themes/` - Primary location for theme CSS files
- `/static/core/css/chat/themes/` - Symlink to the above directory

## Available Themes

1. default.css - A clean, modern chat interface with light colors
2. minimal.css - A simplified, minimalist design 
3. modern.css - A more colorful, contemporary design
