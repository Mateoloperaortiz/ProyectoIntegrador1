About Smart Routing

Smart Routing automatically selects the most appropriate AI tool for your query based on its content. Here's how it works:

Content Analysis

Our system analyzes your message to determine what type of task you're trying to accomplish.

Tool Selection

Based on the analysis, it selects from specialized AI tools including:

  • Text Generator - For general questions and discussions
  • Image Generator - When you want to create or modify images
  • Code Generator - For programming help and code generation
  • Video Generator - For video creation queries
  • Word Processor - For text editing, summarization, or translation
Smart Defaults

If your query doesn't clearly match a specialized tool, the system defaults to the most versatile Text Generator.
